Why is JavaScript popular in web development?
  • JavaScript was the language for Netscape Navigator back when that was the dominant browser.
  • Internet Explorer had (and still has) both JavaScript and VBScript, but obviously if you want to work in more than just IE you have no choice but to use JavaScript.
  • A simple learning curve and flexible skill.
What are the great things about JavaScript?
  • Easy to learn and flexible.
  • Shortcut Notations
  • Native JavaScript Functions regarding Math, Array And String
  • Javascript is executed on the client side.
What are its limitations?
  • Security Issues
  • Javascript rendering varies
  • Can't implement multiprocessing or multithreading
